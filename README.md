# shape-detector
Overview

This is a simple Shape Detection Web App built using HTML, CSS, and JavaScript.
It detects basic geometric shapes such as circle, triangle, rectangle, pentagon, and star from an image.
The app allows you to upload your own image or use preloaded test images.

# Features

Upload an image and automatically detect shapes

Shows shape type, confidence score, area, and position

Displays total processing time

Works completely in the browser (no external libraries needed)

# Steps to Run

Download the project folder or clone it from GitHub

Open the folder

Double-click on index.html → it will open in your web browser

Wait a few seconds for the page to load

Click “Upload Image” or select a test image shown on the page

The app will process the image and show the detected shapes below the canvas

# How It Works

The uploaded image is converted to pixel data

The program analyses edges and shapes

It classifies detected objects into shapes like circle, rectangle, triangle, etc.

It then shows confidence and area for each detected shape

# Technologies Used

HTML5 Canvas API – to draw and read image data

CSS3 – for styling the interface

JavaScript (TypeScript-compatible) – for implementing the shape detection algorithm
